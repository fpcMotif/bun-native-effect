# Security Policy

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Report security issues by emailing the maintainer directly. You should receive a response
within 72 hours. If the issue is confirmed, a patch will be released as soon as possible.

## Scope

This repository is a research workspace. Security reports relevant to the upstream projects
(`oven-sh/bun`, `Effect-TS/effect`) should be reported to their respective security contacts.
